# Operations Checklist

## Ingest
- [ ] Identify and preserve the raw source
- [ ] Add minimal metadata
- [ ] Read the source carefully
- [ ] Update affected wiki pages
- [ ] Update `index.md`
- [ ] Append `log.md`
- [ ] File back any durable synthesis

## Query
- [ ] Read `index.md`
- [ ] Open relevant pages
- [ ] Synthesize from wiki state
- [ ] Cite pages or sources when helpful
- [ ] Save durable answers back into the wiki

## Lint
- [ ] Look for contradictions
- [ ] Look for stale claims
- [ ] Look for orphan pages
- [ ] Look for duplicate concepts
- [ ] Look for missing links
- [ ] Look for provenance gaps
- [ ] Log findings and repairs
