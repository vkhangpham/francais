### Notation des Accents (clavier non-français)
- **`** = accent grave (è, à, ù)
- **'** = accent aigu (é)
- **^** = accent circonflexe (ê, ô, â)
- **:** = tréma (ï, ë)
- **,** = cédille (ç)
