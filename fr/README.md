# Global French Wiki

This directory is the top-level compiled wiki for the entire French learning workspace.

Purpose:
- turn the whole study system into a compounding knowledge base
- compile durable learning from chats, sessions, questions, textbook work, corrections, and review loops
- give the agent one global place to ingest, query, and lint French-learning knowledge
- make `PO` one module inside a broader French wiki, not the only wiki-like area

This layer complements the existing JSON trackers and session journals.

Keep using:
- `06_vocabulaire/vocabulaire_master.json` for machine-readable vocabulary tracking
- `05_grammaire_supplementaire/erreurs_en_cours.json` for machine-readable error tracking
- `data/progression_master.json` for performance/progression tracking
- `sessions/` for event-style session journals

But compile durable reusable knowledge here under `fr/`.

Read first:
- `schema.md`
- `index.md`
- `log.md`
- `reviews/lint_notes.md`

Current maturity: scaffold introduced; global content migration still in progress.
