# PO Pages

Purpose:
- define the `PO` branch of the global French wiki
- gradually integrate the existing `po/` wiki instance into the broader `fr/` architecture

Typical content:
- scenario pages
- framework pages
- concept pages
- speaking-specific syntheses that also link back to shared profile or vocabulary pages

Primary upstream sources:
- existing `po/` wiki
- future PO ingests
- shared learner-profile pages under `fr/profile/`

Initial migration targets:
- link the existing `po/` structure into the global index and migration map
- later decide whether to mirror, migrate, or keep `po/` as a specialized sub-wiki

Status: existing content currently lives mainly in the separate `po/` tree.
