# Global Raw Source Policy

Store source material here when a French-learning event should preserve provenance before or alongside compilation into canonical wiki pages.

Recommended subfolders:
- `chats/`
- `sessions/`
- `books/`
- `prompts/`
- `teacher_feedback/`
- `questions/`
- `transcripts/`
- `draft_archives/`

Naming guidance:
- `YYYY-MM-DD_topic_source-type.md`
- example: `2026-04-06_ce-que-ce-qui-dont_chat-clarification.md`

Rules:
- do not silently rewrite historical source notes
- if understanding changes, add a newer note and update the wiki pages
- keep enough provenance that later sessions know where a fact, explanation, or example came from
- use raw capture for any clarification or exercise result that should influence future sessions beyond a single chat
