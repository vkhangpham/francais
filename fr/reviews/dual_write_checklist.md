# Dual-Write Checklist

Use this after every important French-learning session.

## 1. Preserve the event
- [ ] update or create the session journal / event artifact
- [ ] keep the source book, chapter, pages, or prompt visible in the record

## 2. Update trackers only if justified
- [ ] vocabulary tracker updated if there was real durable vocabulary gain
- [ ] grammar error tracker updated if a stable pattern changed
- [ ] progression tracker updated if the session changed next-step planning

## 3. File durable knowledge into `fr/`
- [ ] choose at least one canonical target page under `fr/`
- [ ] if no page exists yet, create the right page rather than burying the knowledge in the journal
- [ ] move explanations, contrasts, traps, or reusable examples into the canonical page
- [ ] add links to related families when useful

## 4. Preserve provenance when needed
- [ ] if the clarification came from a concrete source or exchange, capture it under `fr/raw/` when useful
- [ ] avoid silent rewriting of historical source notes

## 5. Update navigation
- [ ] update `fr/index.md` if the page set changed
- [ ] append `fr/log.md` if the global wiki changed meaningfully

## 6. Final check
- [ ] ask whether future sessions can now reuse this knowledge without reconstructing it from scratch
