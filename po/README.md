# PO Wiki

This is the first concrete wiki instance built with the general `llm-wiki-maintainer` pattern.

Purpose:
- build a personal, reusable French speaking base
- turn interviews, prompts, revisions, and feedback into a persistent PO wiki
- produce scripts, cue cards, skeletons, and follow-up material that compound over time
- prepare not only personal introductions and short scenarios, but also the more demanding Part 3 opinion/argument tasks through reusable frameworks

This is not a live oral simulation system.
It is a preparation and maintenance system for speaking material you can rehearse out loud by yourself or later use with a teacher.
