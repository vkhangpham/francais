# Self-Introduction Concept

## Purpose

Capture reusable building blocks for self-introduction tasks so future versions do not restart from zero.

## Typical components
- greeting
- name or identity marker
- current situation
- connection to French
- one or two personal traits or interests
- short closing or transition

## Reuse rule

When a new self-introduction version is built, extract reusable lines or structures and link them back here if they can support other tasks.
