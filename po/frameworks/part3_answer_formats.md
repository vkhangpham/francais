# Part 3 Answer Formats

## Purpose

Reusable answer shapes for difficult TCF-style opinion and argument tasks.

## Format A: clear opinion -> two reasons -> short closing

Use when the prompt invites a direct position.

Shape:
1. state your position clearly
2. give first reason
3. give second reason
4. add a quick example or consequence
5. close by restating the main point

Mini-frame:
- À mon avis, ...
- D'abord, ...
- Ensuite, ...
- Par exemple / En effet, ...
- Donc / Finalement, ...

## Format B: advantages -> limits -> position

Use when a topic has obvious pros and cons.

Shape:
1. acknowledge the main advantage
2. introduce the main limit or risk
3. state your final judgment

Mini-frame:
- Il est vrai que ...
- Cependant / En revanche, ...
- Au final, je pense que ...

## Format C: comparison -> criterion -> preference

Use when choosing between options.

Shape:
1. name both options
2. compare them on 1 or 2 criteria
3. explain which one you prefer and why

Mini-frame:
- Les deux options ont des avantages, mais ...
- Si on regarde ..., ...
- Personnellement, je choisirais ... parce que ...

## Format D: problem -> causes -> solution

Use for society, education, work, technology, or environment prompts.

Shape:
1. identify the problem
2. explain one or two causes
3. propose a realistic solution
4. note a condition or limit if useful

Mini-frame:
- Le vrai problème, c'est que ...
- Cela s'explique par ...
- Une solution serait de ...
- Mais pour que cela fonctionne, il faut ...

## Format E: nuanced answer

Use when an extreme answer would sound weak.

Shape:
1. reject oversimplification
2. say "it depends" in a more developed way
3. provide the deciding condition
4. conclude with a balanced position

Mini-frame:
- Je ne dirais pas que ...
- Tout dépend de ...
- Dans certains cas, ... mais dans d'autres, ...
- Donc, il vaut mieux ...

## Practice rule

Do not memorize these as fixed speeches.
Memorize them as reusable skeletons that can receive new vocabulary and examples quickly.
