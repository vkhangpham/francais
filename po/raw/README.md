# Raw Source Policy

Store immutable source material here.

Recommended subfolders:
- `interviews/`
- `prompts/`
- `teacher_feedback/`
- `transcripts/`
- `draft_archives/`

Naming guidance:
- `YYYY-MM-DD_topic_source-type.md`
- example: `2026-04-06_self-introduction_interview.md`

Rules:
- do not silently rewrite historical source notes
- if reality changes, add a new source note and update the wiki pages
- keep enough provenance that later sessions know where a fact or wording came from
