**3.4** : cela

### Exercice 4 : Distinction "celui-ci" / "celui-là"
**4.1** : celui-ci  
**Explication** : Objet proche → "ci"

**4.2** : celui-ci  
**Explication** : Dernier nommé (Pierre) → "ci"

**4.3** : celui-là  
**Explication** : Objet éloigné → "là"

**4.4** : celle-là  
**Explication** : Premier élément mentionné (Marie) → "là"

**4.5** : celle-ci  
**Explication** : Simple préférence → généralement "ci"

**4.6** : celui-là  
**Explication** : Registre familier péjoratif → souvent "là"

### Exercice 5 : Correction d'erreurs
**5.1** : J'ai plusieurs stylos. Celui-ci marche bien / Celui de droite marche bien  
**Explication** : "Celui" ne peut être employé seul

**5.2** : Tes chaussures et celles de ton frère sont sales  
**Explication** : "Chaussures" est féminin pluriel → celles

**5.3** : Cette voiture ? Celle-ci va très vite / Elle va très vite  
**Explication** : "Celle" ne peut être employé seul

**5.4** : Il a parlé de ses projets. Ceux-ci l'intéressent beaucoup  
**Explication** : "Projets" est masculin pluriel → ceux-ci

**5.5** : Entre ces deux robes, celle-ci est plus jolie  
**Explication** : Forme composée nécessaire

**5.6** : Cela ne saurait convenir dans ce contexte  
**Explication** : Lettre formelle → "cela" (registre soutenu)
